## 개요
나무마크 기반의 파이썬 위키 엔진 입니다. (파이썬 3.5 이상)

## 클론 명령어
### 일반
 * `git clone -b stable https://github.com/2DU/openNAMU.git`

### 베타
 * `git clone -b master https://github.com/2DU/openNAMU.git`

## 설치법
 * [참조](http://namu.ml/w/오픈나무%2F설치법)
 
## set.json 설명
 * db = 데이터베이스 이름

set.json를 삭제하면 다시 새로 만들 수 있습니다.
